# NServiceBusContrib.Persistence.Sqlite

A community SQLite persistence package for NServiceBus 10. Provides Outbox, Saga, Subscription, and synchronized storage session support, backed by `Microsoft.Data.Sqlite`.

This project is not affiliated with or endorsed by Particular Software. NServiceBus is a registered trademark of Particular Software.

For TransactionalSession support, install the `NServiceBusContrib.Persistence.Sqlite.TransactionalSession` package.

See the project repository for documentation and examples.
